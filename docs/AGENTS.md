# Agents IA

Ce projet utilise un système d'agents IA pour automatiser le développement logiciel.

Les agents sont capables d'analyser le code existant, d'implémenter de nouvelles fonctionnalités, de corriger des bugs et de créer des commits, en suivant les conventions et l'architecture du projet.

Ils opèrent dans des worktrees Git isolés afin de travailler en parallèle sans interférer les uns avec les autres.
